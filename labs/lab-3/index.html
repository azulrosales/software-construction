Preguntas

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?

La regla !important en CSS se utiliza para darle prioridad a una declaración de estilo sobre otras declaraciones. Se debe tener cuidado al utilizar esta regla, ya que puede hacer que el código sea más difícil de mantener y de entender.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?

Es importante escoger cuidadosamente la imagen de fondo que se utiliza en una página HTML, ya que esta puede afectar la legibilidad y la accesibilidad de la página, así como la experiencia del usuario en general.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?

En general, se recomienda utilizar unidades relativas como porcentajes o ems cuando sea posible, ya que permiten que los elementos de la interfaz de usuario se ajusten automáticamente al tamaño del contenedor o de la pantalla. Sin embargo, en ciertas situaciones, como la definición del tamaño de fuente, puede ser más apropiado utilizar una unidad absoluta como píxeles o puntos para lograr la consistencia y precisión deseada.
La elección de las unidades de estilo dependerá del contexto específico y de las necesidades del proyecto.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?

Porque reduce el tamaño del archivo CSS, lo que disminuye el tiempo de carga y mejora la velocidad de la página, reduce la cantidad de datos que se deben transferir desde el servidor al navegador del usuario, reduce el número de solicitudes HTTP necesarias para cargar la página y puede mejorar la eficacia de la caché del navegador.
