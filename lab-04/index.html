Descripción del problema

Una biblioteca necesita una herramienta para gestionar sus libros. Cada libro tiene un título, un autor y una categoría. La biblioteca necesita poder agregar nuevos libros, buscar libros por título o autor, y mostrar todos los libros ordenados por categoría.

Solución

crearemos un objeto llamado "Library" que contendrá una matriz de objetos "Book", cada uno con sus propiedades de título, autor y categoría. El objeto Library tendrá tres métodos:

  • addBook: Para agregar un nuevo libro a la matriz.
  • searchBook: Para buscar un libro por título o autor.
  • displayBooks: Para mostrar todos los libros ordenados por categoría.


Preguntas

¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Java es un lenguaje de programación orientado a objetos utilizado principalmente para crear aplicaciones empresariales, mientras que JavaScript es un lenguaje de scripting utilizado principalmente en el desarrollo web para agregar interactividad y funcionalidad a las páginas web. Ambos lenguajes comparten algunas similitudes en la estructura de los programas y en la orientación a objetos, pero difieren en su sintaxis, tipado y plataformas de uso.

¿Cómo se declara una variable con alcance local dentro de una función?

Se utiliza la palabra clave let o const dentro del bloque de la función donde se quiere declarar la variable.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Utilizar variables globales dentro de funciones puede generar confusión, errores, interferencia, sobrescritura accidental y dificultad para razonar sobre el código. Se recomienda evitarlo y utilizar la encapsulación y la modularización para mantener el código limpio y organizado.
